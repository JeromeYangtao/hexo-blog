#1.什么是Markdown?
Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档。

#2.为什么要用Markdown?
总结 Markdown 的优点如下：

  • 纯文本，所以兼容性极强，可以用所有文本编辑器打开。
  • 让你专注于文字而不是排版。
  • 格式转换方便，Markdown 的文本你可以轻松转换为 html、电子书等。
  • Markdown 的标记语法有极好的可读性。

同时Markdown语法简单，使用方便，学习所需时间很短，尤其是对于前端而言。
